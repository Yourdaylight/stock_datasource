# 龙虎榜追踪系统功能说明

## 功能概述

龙虎榜追踪系统是一个全面的股票交易异动监控和分析平台，提供实时的龙虎榜数据分析、投资组合风险评估和市场异动预警功能。

## 主要功能模块

### 1. 龙虎榜数据查询
- **实时数据**: 获取每日龙虎榜上榜股票信息
- **历史查询**: 支持按日期、股票代码查询历史龙虎榜记录
- **多维筛选**: 支持按涨跌幅、成交额、上榜原因等条件筛选
- **机构席位**: 查看机构专用席位的买卖情况

### 2. 智能分析服务
- **席位集中度分析**: 计算龙虎榜席位的集中度指数和风险等级
- **资金流向分析**: 分析股票的资金流入流出趋势和稳定性
- **交易模式识别**: 识别异常交易模式和投资行为特征
- **风险评估**: 基于多维度指标进行综合风险评估

### 3. 投资组合龙虎榜分析
- **持仓监控**: 实时监控投资组合中股票的龙虎榜状态
- **风险预警**: 识别高风险持仓并提供预警信息
- **资金流向**: 分析持仓股票的整体资金流向趋势
- **投资建议**: 基于龙虎榜数据提供个性化投资建议

### 4. 异动预警系统
- **实时监控**: 7x24小时监控市场异动情况
- **多级预警**: 提供高、中、低三级风险预警
- **智能筛选**: 自动识别成交量异动、价格异常波动等情况
- **及时通知**: 重要异动情况及时推送预警信息

### 5. 市场统计分析
- **整体概况**: 提供龙虎榜市场的整体统计数据
- **行业分布**: 分析不同行业在龙虎榜中的分布情况
- **上榜原因**: 统计各种上榜原因的分布和趋势
- **趋势分析**: 展示龙虎榜数据的历史趋势变化

## 技术架构

### 后端服务
- **数据插件**: 基于Tushare API的龙虎榜数据采集插件
- **分析服务**: 席位集中度分析、异动检测等核心算法服务
- **智能代理**: 基于AI的龙虎榜分析和投资建议生成
- **API接口**: RESTful API提供数据查询和分析功能

### 前端界面
- **数据表格**: 响应式龙虎榜数据展示和筛选界面
- **分析图表**: 可视化的分析结果展示
- **投资组合集成**: 与现有投资组合管理系统无缝集成
- **实时更新**: 支持数据的实时刷新和动态更新

### 数据存储
- **龙虎榜数据表**: 存储每日龙虎榜基础数据
- **机构席位表**: 存储机构专用席位交易数据
- **分析结果缓存**: 缓存分析结果提高查询效率

## 使用指南

### 1. 访问龙虎榜页面
- 在主导航栏点击"龙虎榜分析"进入主页面
- 或直接访问 `/toplist` 路径

### 2. 查看龙虎榜数据
- 选择查询日期查看当日龙虎榜数据
- 使用筛选器按股票代码、涨跌幅等条件筛选
- 点击"详情"查看股票的详细龙虎榜信息
- 点击"分析"获取股票的深度分析报告

### 3. 投资组合分析
- 在投资组合页面切换到"龙虎榜分析"标签
- 查看持仓股票的龙虎榜状态和风险评估
- 关注系统提供的风险预警和投资建议

### 4. 异动预警
- 在异动预警页面查看最新的市场异动情况
- 按风险级别筛选查看不同程度的预警信息
- 点击预警项目查看详细的异动分析

### 5. 市场统计
- 查看龙虎榜市场的整体统计数据
- 分析不同行业和上榜原因的分布情况
- 了解市场整体的资金流向趋势

## API接口说明

### 主要接口
- `GET /api/toplist/date/{date}` - 获取指定日期龙虎榜数据
- `GET /api/toplist/stock/{ts_code}/history` - 获取股票龙虎榜历史
- `GET /api/toplist/analysis/stock/{ts_code}` - 获取股票分析报告
- `POST /api/toplist/portfolio/analyze` - 分析投资组合龙虎榜情况
- `GET /api/toplist/alerts` - 获取异动预警信息
- `GET /api/toplist/stats` - 获取市场统计数据

### 数据格式
所有API返回JSON格式数据，包含标准的响应结构：
```json
{
  "success": true,
  "data": {...},
  "message": "操作成功",
  "total": 100,
  "page": 1,
  "page_size": 50
}
```

## 配置说明

### 数据源配置
- 在 `src/stock_datasource/plugins/tushare_top_list/config.json` 中配置Tushare API
- 设置数据更新频率和缓存策略

### 分析参数配置
- 席位集中度阈值设置
- 异动检测敏感度参数
- 风险评估权重配置

### 预警规则配置
- 自定义异动检测规则
- 设置预警通知方式
- 配置风险等级判断标准

## 注意事项

1. **数据延迟**: 龙虎榜数据通常在交易日收盘后更新，存在一定延迟
2. **API限制**: 受Tushare API调用频率限制，避免过于频繁的数据请求
3. **风险提示**: 分析结果仅供参考，投资决策需结合多方面因素考虑
4. **数据准确性**: 定期校验数据准确性，如发现异常及时处理
